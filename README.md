# Morabaraba 3D Human vs AI

This is a static GitHub Pages version of the original C++ Morabaraba project.

- Human plays as `X`.
- AI plays as `O`.
- `Alg1` is the Hard AI.
- `Alg2` is the Medium AI.
- The board is rendered in 3D on a table in a park.
- Impossible moves, wins, and losses trigger animations.
- The original C++ files are still included unchanged.

## Run Locally

Open `index.html` in a browser with internet access so the Three.js module can load.

## Host On GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Go to **Pages**.
4. Set the source to the main branch and the repository root.
5. Save, then open the GitHub Pages URL after it finishes publishing.
