Overview
This project is a C++ implementation of the traditional Southern African strategy board game **Morabaraba**, developed as part of the **ELEN2020/ELEN2004 Software Development I** course at the **University of the Witwatersrand**.
The program simulates a complete Morabaraba game where **two algorithms (Alg 1 and Alg 2)** compete against each other automatically without user interaction. The game follows the project specification provided in the course brief. :contentReference[oaicite:0]{index=0}



Features

- Full Morabaraba board implementation (24 positions)
- Automatic gameplay between two AI algorithms
- Placement phase
- Movement phase
- Mill detection
- Automatic removal of opponent pieces after forming a mill
- Win and draw detection
- Reads game configurations from an input file
- Writes complete game history to an output file
- Object-oriented implementation using C++
